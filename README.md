Overview

Most password generators rely on mathematical formulas — deterministic systems that, in theory, can be reverse-engineered.

This project takes a fundamentally different approach.

Each character in your password is generated through a real quantum measurement. A qubit is placed into superposition and collapses according to the laws of physics.

No formulas.
No patterns.
No seeds.
Only true quantum randomness.

How It Works
A quantum circuit is initialized with 7 qubits
Each qubit is placed into superposition (both 0 and 1 simultaneously)
Measurement collapses each qubit into a definite state (0 or 1)
The resulting 7-bit value maps to a character from a 70-character set
The process repeats until the password is complete
